# Navigation Setup Guide - Formula S Style Dropdowns

Your theme now has enhanced dropdown navigation just like the Formula S website! Here's how to set it up in Shopify.

## How to Create Dropdown Navigation Menus in Shopify

### Step 1: Access Navigation Settings

1. Log in to your **Shopify Admin**
2. Go to **Online Store > Navigation**
3. Click on **Main menu** (or create a new menu)

### Step 2: Create Main Menu Items

Based on the Formula S website, here are recommended categories for an automotive parts store:

**Main Navigation Items:**
1. Special Projects
2. Accessories
3. Appearance
4. Brakes
5. Drivetrain
6. Engine
7. Electronics
8. Kits
9. Suspension
10. Wheels
11. Services

### Step 3: Add Dropdown Items (Sub-menus)

To create dropdowns, you need to add **nested menu items** under each main category.

#### Example: "Engine" Category with Dropdowns

1. Click **Add menu item**
2. **Name:** Engine
3. **Link:** /collections/engine-parts
4. Click **Add**

5. Now add sub-items by clicking **Add menu item** again:
   - Indent the item by dragging it slightly to the right under "Engine"
   - This creates the dropdown!

**Example Engine Dropdown Items:**
- Air Intake Systems
- Exhaust Systems
- Turbo & Superchargers
- Engine Internals
- Cooling Systems

- Engine Mounts

### Step 5: Assign Menu to Theme

1. After creating your navigation structure, **Save** the menu
2. Go to **Online Store > Themes**
3. Click **Customize** on your active theme
4. In the theme editor, find the **Header** section
5. Under "Menu", select your newly created menu
6. Click **Save**

## Tips for Best Results

1. **Keep dropdown items organized** - Group related products together
2. **Use clear names** - Make it easy for customers to find what they need
3. **Limit dropdown length** - Aim for 5-10 items per dropdown for best UX
4. **Link to collections** - Each dropdown item should link to a relevant collection or page
5. **Test on mobile** - Make sure navigation works well on all devices

## Formula S Style Menu Recommendations

For an authentic Formula S experience, use these category names:

- **Special Projects** - Custom builds and featured projects

- **Appearance** - Body kits, styling parts

- **Drivetrain** - Transmission, differential, axles

- **Services** - Installation, tuning, consultation
