You have **3 layout options**:

**Option 1: Logo Left, Nav Right** (Default - Like Formula S)

- Classic ecommerce layout

### Sticky Header

**Enable/Disable:**
- Toggle "Sticky Header" checkbox
- **ON** = Header follows you as you scroll
- **OFF** = Header stays at top of page

### Transparent Header

**Enable/Disable:**
- Toggle "Transparent Header" checkbox
- **ON** = Header overlays your hero banner
- **OFF** = Solid black header
- Automatically turns solid when scrolling

Perfect for dramatic hero images!

## ðŸ”§ Troubleshooting

### "Header looks weird"
- Make sure you removed the old header first
- Only have ONE header section active
- Clear browser cache

### "Search doesn't work"
- Search overlay is automatic
- Make sure JavaScript is enabled
- Try clicking the search icon again

### "Mobile menu won't open"
- Check browser console for errors
- Make sure you saved the theme
- Try refreshing the page

### "Dropdowns don't show"
- Make sure menu items are indented (see NAVIGATION-SETUP-GUIDE.md)
- Hover properly over parent item
- Check that menu is assigned to header

### "Transparent header doesn't work"
- Only works with hero banners
- Needs to be on homepage/pages with banners
- Turns solid when scrolling (this is intentional)
